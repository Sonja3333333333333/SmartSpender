# Smart Spend

**Smart Spend** — це зручний мобільний застосунок для обліку особистих витрат, який допомагає контролювати щоденні витрати та аналізувати власний бюджет.

Застосунок поєднує швидку фіксацію покупок, категоризацію витрат та наочну статистику. Завдяки цьому користувачі можуть ефективніше керувати своїми грошима, виявляти *“фінансові діри”* та швидше накопичувати кошти.

---

## Основні можливості

- **Швидкий запис витрат**  
  Додавання суми та вибір категорії за лічені секунди.

- **Категоризація**  
  Зручний поділ витрат на групи (Продукти, Транспорт, Розваги тощо).

- **Візуальна статистика**  
  Наочна кругова діаграма для аналізу бюджету за місяць.

- **Історія транзакцій**  
  Повний перелік усіх покупок з можливістю редагування.

- **Offline-режим**  
  Стабільна робота без доступу до інтернету з локальним зберіганням даних.

---

## Мета проєкту

Створити простий та сучасний інструмент для фінансової дисципліни, який автоматизує облік щоденних витрат.

Проєкт покликаний:

- зменшити стрес від непередбачуваних витрат  
- допомогти користувачам зрозуміти свої фінансові звички  
- зробити процес ведення бюджету максимально легким і доступним для кожного
